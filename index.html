Quiz app

Code Quiz

The code quiz is a simple app that asks questions and gets user responses via multiple choice buttons which will responds with green(correct) or red (incorrect) and then move to the next question until the quiz is complete. When the quiz is complete the app will prompt the user for their initials and the results will be saved to local storage so they can come back later and view them. The code quiz also has a timer which counts down as the user is attempting the quiz and responds to correct or incorrect answers by adding or deducting time from the timer, this is also saved at the end.
